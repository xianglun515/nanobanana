📝 测试说明

下载功能改进：

  • Data URL 格式： 直接使用 link.download 下载
  • HTTP URL 格式： 使用 fetch 获取图片数据，然后创建 blob URL 下载
  • 避免跳转： 不再直接使用 link.href = imageUrl，避免浏览器跳转
  • 错误处理： 包含网络错误重试机制
  • 内存管理： 自动清理 blob URL 防止内存泄漏
